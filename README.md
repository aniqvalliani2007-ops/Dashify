# 📊 Dashify: Modern AI-Powered Analytics Studio

Dashify is a premium, highly responsive web application that transforms raw spreadsheet data (CSV, Excel) into interactive, publication-ready visual dashboards and intelligent AI-generated insights. Built as a key showcase for my portfolio, Dashify demonstrates modern frontend engineering, robust client-side state management, and seamless integration with third-party AI APIs.

[Live Demo](https://github.com/aniqvalliani2007-ops/Dashify) • [Key Features](#-key-features) • [Tech Stack](#-tech-stack) • [Quick Start](#-quick-start)

---

## 🚀 Key Features

* **📂 Smart File Parsing**: Drag-and-drop CSV (via PapaParse) and multi-sheet Excel files (via SheetJS) parsed instantly in the browser.
* **🗺️ Visual Column Mapping**: Dynamically map columns to X/Y coordinates and automatically detect data types (Numeric, Categorical, Dates).
* **📈 Interactive Sandbox**: Switch between Bar, Line, Pie, Scatter, and Area charts. Apply live aggregations (Sum, Avg, Count, Min, Max) client-side.
* **🤖 AI Data Analyst**: A conversational chat assistant powered by OpenRouter (Gemini, Llama) that analyzes your dataset context and answers queries.
* **🧠 Chart Recommendations**: Automated, smart layout recommendations generated by AI, with local heuristic rules as a robust offline fallback.
* **🔐 Hybrid Auth & Persistence**: Supabase integration for user sessions and database storage, with an automated local storage fallback demo mode.

---

## 🛠️ Tech Stack

* **Frontend**: React 19, Vite, Tailwind CSS v4, React Router 7
* **Data Visualization**: Recharts, Lucide Icons
* **Backend & Database**: Supabase (PostgreSQL), Local Storage Fallback
* **Spreadsheet Engines**: PapaParse (CSV), XLSX / SheetJS (Excel)
* **AI Integrations**: OpenRouter API (Gemini & Llama models with automatic cascade fallbacks)

## ⚙️ Environment Configuration

To run Dashify locally, create a `.env.local` file in the root directory:

```bash
# Supabase Configuration (Optional; defaults to Demo Mode if missing)
VITE_SUPABASE_URL=https://your-project-id.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key

# OpenRouter AI API Key
VITE_OPENROUTER_API_KEY=your-openrouter-key
```

## 📦 Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/aniqvalliani2007-ops/Dashify.git
   cd Dashify
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```
